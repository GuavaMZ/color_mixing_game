# Changelog

All notable changes to Color Lab will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-12

### Added - Initial Release
- **4 Game Modes**
  - Classic Mode: 300 progressive levels
  - Time Attack: Race against the clock
  - Color Echo: Random challenging targets
  - Chaos Lab: Unstable laboratory with random events

- **Core Gameplay**
  - Intuitive color mixing mechanics (RGB + White/Black)
  - Match percentage indicator
  - Drop limit system
  - Blind mode challenges

- **Progression System**
  - 3-star rating per level
  - Coin economy
  - Unlockable achievements
  - Statistics tracking
  - Combo system for consecutive perfect matches

- **Customization**
  - Lab Upgrade Hub with 20+ customization items
  - Multiple beaker skins
  - Customizable backgrounds, lighting, and surfaces
  - Color gallery with discovery system

- **Helpers & Power-ups**
  - Undo: Reverse last drop
  - Extra Drops: Increase drop limit
  - Help Drop: Auto-add needed color
  - Reveal Color: Temporarily show hidden color in blind mode

- **Accessibility**
  - 4 languages: English, Arabic, Spanish, French
  - Colorblind mode support
  - High contrast option
  - Reduced motion mode
  - Haptic feedback (customizable)

- **Visual & Audio**
  - Cosmic laboratory theme
  - Dynamic particle effects
  - Adaptive background music
  - Sound effects with volume controls

- **Performance**
  - Auto-detection of low-end devices
  - Performance optimization mode
  - Object pooling for particles
  - Smooth 60 FPS gameplay

### Technical
- Built with Flutter & Flame engine
- Offline-first design
- Local save system with SharedPreferences
- Portrait-only orientation
- Android 5.0+ (API 21) support
